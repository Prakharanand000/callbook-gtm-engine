## What this is

An end-to-end GTM system that finds, scores, and generates personalized outreach for Callbook.ai's ideal customers — lenders actively hiring for collections and debt recovery roles.

**Live demo:** https://callbook-gtm-engine.streamlit.app/

---

## How it works

```
Apify LinkedIn Jobs Scraper
        ↓
Apify LinkedIn Company Enrichment
        ↓
Python Scoring Pipeline (score_leads.py)
        ↓
Streamlit Dashboard (dashboard.py)
        ↓
Outreach Templates + KugelAudio Voice Notes
```

### Scoring logic (100-point scale)

| Signal | Weight | Description |
|--------|--------|-------------|
| Job title seniority | 40% | VP/Director = 40pts, Manager = 30pts, Specialist = 20pts |
| Company size | 25% | 50–600 employees = sweet spot |
| Industry fit | 25% | Fintech/consumer lending = top tier |
| Hiring recency | 10% | Posted ≤5 days = urgent |

### Tier cutoffs
- **Tier A (≥75):** Contact within 24 hours
- **Tier B (50–74):** Add to nurture sequence
- **Tier C (<50):** Monitor, deprioritize

## Setup

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. (Optional) Run Apify pipeline and place outputs in data/raw/
#    See "Apify Pipeline Guide" tab in the dashboard

# 3. Run the scoring script
python score_leads.py

# 4. Launch the dashboard
streamlit run dashboard.py
```

The dashboard works out of the box with built-in demo data — no Apify run required to see it working.

## Outreach sequence

| Day | Channel | Action |
|-----|---------|--------|
| 0 | LinkedIn | Connection request |
| 2 | LinkedIn DM | Template C (short) |
| 5 | Email | Template A or B |
| 8 | KugelAudio | 30-sec voice note |
| 14 | Email | One-line follow-up bump |

## Hackathon context

Callbook.ai is an AI platform for lenders to understand borrowers, decide the right next action, and recover more. Bootstrapped to $1M ARR in 8 months.

This GTM system targets their ICP: VP/Director of Collections at fintech/consumer lending companies (50–2,000 employees) actively scaling their collections operations.

The key insight: **hiring signals are buying signals.** A company posting "Collections Manager" is in pain Callbook solves, right now.
